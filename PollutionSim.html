Pollution Sim

Build powerplants and factories, get money and energy, but don't overpollute the planet or you will lose!

This is a 2D topdown simulation project, mainly being done to become more familiar with Unity Game Engine


This project is being worked on as a group in my Modeling and Simulation Applications class with some other teammates.





Version History

The letter prefacing the numbers in a version mean something.

v = pre-public-release testing version, r = public release, a = alpha release, s = restricted public release
